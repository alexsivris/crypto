uint8_t ctr ( CONTEXT const  ctx)

Encrypts/Decrypts data in in counter mode. The structure |ctx| holds the relevant data. Note that operation happens in place, so input data is overridden by output!

Parameters
ctxThe encryption/decryption context.
Returns
Returns 0, if encryption/decryption was successful
